# Task Management App

A full-stack task management application with JWT authentication, built as part of a technical assignment.

## Tech Stack

**Backend:** NestJS, MongoDB (Mongoose), JWT Authentication, class-validator
**Frontend:** React, TypeScript, TanStack Query, Tailwind CSS

## Project Structure
task-management-app/
├── backend/          # NestJS REST API
└── frontend/         # React + TypeScript client

## Features

- User registration & login (JWT-based auth, HTTP-only cookies)
- Create, read, update, delete tasks
- Mark tasks as completed
- Filter tasks by status (Pending / Completed)
- Responsive UI (Tailwind CSS)
